# i18n / ローカライゼーション

## msg() 関数

キーベースの多言語メッセージ関数。`LANGUAGE` 変数で言語を切り替える。

- 対応言語: `ja`（日本語）、`en`（英語、デフォルト）
- キー形式: `"<phase>.<action>"` （例: `"dev.success"`, `"review.lgtm"`, `"analysis.split"`）
- 動的テキスト: `%s` プレースホルダーを使い、呼び出し側で `printf` する

```bash
# 使用例
comment_body="🤖 Dev: $(printf "$(msg "dev.success")" "$pr_url")"
```

## 新しいメッセージの追加

1. `msg()` 関数の `ja` と `en` 両方の case に追加する
2. キーはフェーズ名をプレフィックスにする
3. 未知のキーはそのまま返す（フォールバック）

## summarize_failure() 関数

Claude を使って失敗原因を要約する関数。

- 入力: Claude 出力（末尾30行、最大2000文字）とコンテキスト文字列
- 出力: LANGUAGE で指定された言語での要約（3-5行）
- フォールバック: Claude 呼び出し失敗時は `"(auto-summary unavailable)"`
